## Требования:
- `Mobile-first`.
- НЕ использовать бутстрап. 
- Можно верстать с использованием `flexbox`.
- НЕ использовать `grid css`.
- НЕ использовать плагины конструкторов шаблонов.
- НЕ использовать `Jquery`.
